# Drone ReID Tracking with PX4 + ROS2

## Overview
Jetson 기반 AI(ReID)를 이용하여 드론이 사람을 따라다니는 시스템 구축

## System Architecture
Jetson (AI)
→ MAVLink
→ PX4 (Pixhawk)
→ Drone Control

## Goal
- Real-time person tracking
- Autonomous following
- Onboard decision-making
